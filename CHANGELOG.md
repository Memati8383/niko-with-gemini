# Değişiklik Günlüğü

Tüm önemli değişiklikler bu dosyada belgelenecektir.

Format [Keep a Changelog](https://keepachangelog.com/tr/1.0.0/) standardına dayanır,

## [1.0.0] - 2026-02-07

### Eklenenler

- 🎉 İlk stabil sürüm yayınlandı
- 🤖 Ollama entegrasyonu ile çoklu LLM desteği
- 🔐 JWT tabanlı unified authentication sistemi
- 📱 Android native mobil uygulama
- 💻 Modern web chat arayüzü
- 🎤 Sesli komut desteği (mobil)
- 🌐 Cloudflare tunnel entegrasyonu
- 📊 Admin paneli ve kullanıcı yönetimi
- 💾 Sohbet geçmişi kaydetme ve yönetimi
- 🎭 Kişilik modları (Normal, Agresif, Romantik, vb.)
- 🔍 Web arama entegrasyonu (DuckDuckGo)
- 🔄 Otomatik güncelleme sistemi (mobil)
- 📞 Sistem entegrasyonları (arama, WhatsApp, müzik)
- 🎨 Glassmorphism UI tasarımı
- 🌙 Karanlık tema desteği
- 📱 PWA desteği (web)
- 🔊 Text-to-Speech (Edge TTS)
- 📝 Markdown desteği
- 🎯 Düşünce akışı görüntüleme
- 🛠️ Sistem başlatıcı (sistemi_baslat.bat)
